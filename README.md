Projeto BellaVitta

Site de um Restaurante Italiano, criado para fins de aprendizagem.

Descrição

O BellaVitta é um projeto web desenvolvido para simular o site de um restaurante italiano.
Ele apresenta páginas de cardápio, cadastro, comunicados, reservas e pedidos interativos, com design responsivo.(Apenas o FRONT-END)

Tecnologias utilizadas

HTML5

CSS3

JavaScript (básico)

Funcionalidades:

Página inicial com apresentação do restaurante.

Cardápio com pratos e imagens ilustrativas.

Sistema de pedidos com carrinho de compras dinâmico.

Design responsivo para diferentes tamanhos de tela.

Objetivo

Projeto desenvolvido para fins educacionais, com foco em front-end e experiência do usuário (UX).
